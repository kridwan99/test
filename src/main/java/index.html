DELI-cious Sandwich Shop POS

Capstone Project 2 | Java CLI Application

Overview

DELI-cious is a command-line point-of-sale system designed for sandwich shops. It provides an interactive menu for building custom sandwiches, choosing drinks, chips, and finalizing orders with receipt generation and file saving.

Features

  • Home Menu: Start Order, View Receipts, Exit
  • Build custom sandwiches with meats, cheese, sauces, toppings
  • Select drink size & flavor
  • Select chip type
  • Signature sandwich options (BLT or Philly)
  • Receipts displayed & saved to file
  • Full object-oriented Java architecture with classes, interfaces, and abstraction

Flowchart

Code Highlights

The application uses principles of Object-Oriented Programming:

  • Encapsulation: Clean separation of UI and business logic
  • Interfaces: SandwichItem, Drink, Chips implement OrderItem
  • Abstraction: Abstract classes for menu management
  • Lambdas: Used in receipt formatting and filtering
